# Portfolio – Samuel Okotako

Sito personale statico per presentare profilo, competenze e progetti in **Data Management, Machine Learning e Cloud AWS**.  
Design dark elegante, completamente **bilingue (IT/EN)** e responsive.

---

## Tecnologie
- **HTML5**, **CSS3**, **JavaScript**
- Layout responsive (senza framework)
- Switch lingua IT/EN con `localStorage`
- Hosting compatibile: GitHub Pages, apache2, Netlify, Vercel, AWS S3/CloudFront

---

## Struttura
├── index.html
├── style.css
└── script.js
